**文档类型**：UI/UX 设计规范 + 线框说明  
**关联文档**：[PDD 楔子版](PDD-新婚备育家庭财务规划教练.md)  
**文档版本**：V1.0  
**撰写日期**：2026-06-11  
**设计状态**：MVP 原型定稿  
**可交互原型**：[`../prototype/index.html`](../prototype/index.html)（浏览器打开，模拟 375px 小程序视口）

---

## 一、设计原则

| 原则 | 说明 |
|------|------|
| **温暖可信** | 家庭财务场景，避免银行 App 的冷蓝/金色；用暖色底 + 绿色表达「健康/安全」 |
| **3 秒懂价值** | 首屏必须回答「这是什么、对我有什么用」 |
| **少输入多反馈** | 向导每步 ≤ 3 个字段；实时显示占比/警告 |
| **报告即产品** | 规划书页面视觉权重最高，健康分是「成就符号」 |
| **轻量追踪** | 看板一屏可读；周度填报 7 个数字，无复杂表单 |

**参考气质**：Notion 的清晰 + 宝宝树/小红书备育内容的温暖感，**不要**传统记账 App 的表格堆砌感。

## 二、设计系统（Design Tokens）

### 2.1 色彩

| Token | 色值 | 用途 |
|-------|------|------|
| `--color-primary` | `#2D8B7A` | 主色：按钮、进度条正常态、链接 |
| `--color-primary-light` | `#E8F5F2` | 主色浅底：选中卡片、标签背景 |
| `--color-accent` | `#E8956A` | 强调 CTA、备育模块点缀 |
| `--color-bg` | `#FBF9F7` | 页面背景（暖白） |
| `--color-surface` | `#FFFFFF` | 卡片表面 |
| `--color-text` | `#1A1A1A` | 主文字 |
| `--color-text-secondary` | `#6B7280` | 次要说明 |
| `--color-text-tertiary` | `#9CA3AF` | 占位符、辅助 |
| `--color-border` | `#EDE9E4` | 分割线、边框 |
| `--color-success` | `#4CAF7D` | 健康/进度 < 70% |
| `--color-warning` | `#F5A623` | 警告 70–90% |
| `--color-danger` | `#E85D5D` | 超支 / 高风险 |
| `--color-score-high` | `#2D8B7A` | 健康分 ≥ 75 |
| `--color-score-mid` | `#F5A623` | 健康分 50–74 |
| `--color-score-low` | `#E85D5D` | 健康分 < 50 |

**对比度**：正文与背景 ≥ 4.5:1；主按钮白字 on `#2D8B7A` 通过 WCAG AA。

### 2.2 字体（小程序）

| 层级 | 字号 | 字重 | 行高 | 场景 |
|------|------|------|------|------|
| H1 | 22px | 600 | 1.35 | 落地页主标题 |
| H2 | 18px | 600 | 1.4 | 页面标题、卡片标题 |
| H3 | 16px | 600 | 1.4 | 区块标题 |
| Body | 15px | 400 | 1.5 | 正文 |
| Caption | 13px | 400 | 1.45 | 说明、标签 |
| Number XL | 48px | 700 | 1 | 健康分大号 |
| Number LG | 24px | 600 | 1.2 | 金额强调 |

**字体栈**：`-apple-system, "PingFang SC", "Helvetica Neue", sans-serif`

### 2.3 间距与圆角

| Token | 值 | 用途 |
|-------|-----|------|
| `--space-xs` | 4px | 图标与文字间距 |
| `--space-sm` | 8px | 紧凑元素 |
| `--space-md` | 16px | 卡片内边距、页面左右边距 |
| `--space-lg` | 24px | 区块间距 |
| `--space-xl` | 32px | 大区块 |
| `--radius-sm` | 8px | 输入框、小按钮 |
| `--radius-md` | 12px | 按钮、标签 |
| `--radius-lg` | 16px | 卡片 |
| `--radius-full` | 999px | 胶囊按钮、进度条 |

### 2.4 阴影

| 级别 | 值 | 用途 |
|------|-----|------|
| 卡片 | `0 2px 12px rgba(26,26,26,0.06)` | 普通卡片 |
| 浮层 | `0 8px 24px rgba(26,26,26,0.12)` | 底部弹层、付费遮罩 |

### 2.5 图标

- 风格：**线性图标**，1.5px 描边，圆角端点
- 尺寸：导航 24px，列表 20px，装饰 32px
- MVP 原型使用 Unicode/SVG 简化；正式开发建议 **Remix Icon** 或微信内置 iconfont

## 三、组件库

### 3.1 按钮

| 类型 | 样式 | 高度 | 场景 |
|------|------|------|------|
| Primary | 填充 `#2D8B7A`，白字 | 48px | 主 CTA |
| Secondary | 白底 + 主色边框 | 48px | 次操作 |
| Ghost | 透明 + 主色字 | 44px | 文字链级 |
| Disabled | 灰底 `#E5E7EB` | 48px | 未完成必填 |

**触控区域**：最小 44×44px。

### 3.2 输入

| 类型 | 说明 |
|------|------|
| 金额输入 | 右对齐数字，前缀 `¥`，大号 18px |
| 城市选择 | 底部 Sheet -picker |
| 阶段卡片 | 三选一，选中态：主色浅底 + 2px 主色边框 |
| 滑块 | 收入快测用，步进 1000 |

### 3.3 进度条

```
轨道高度 8px，圆角 full
填充色按阈值：绿 / 黄 / 红
右侧显示百分比 + 可选 emoji 状态（⚠️ / 正常）
```

### 3.4 健康分环

- 直径 120px（预览页）、160px（报告封面）
- 环宽 10px
- 中央：分数 + 「分」小字
- 颜色随分数段变化

### 3.5 建议卡片

```
┌─ 左侧 4px 色条（按类型：警告黄/提示绿）
├─ 标题 16px 600
├─ 描述 14px secondary
├─ 行动列表（bullet）
├─ 预估影响 caption
└─ [已采纳] [稍后] [忽略] 三按钮横排
```

### 3.6 付费遮罩（Blur Lock）

- 被锁内容：`filter: blur(6px)` + 半透明遮罩
- 中央：锁图标 + 「开通 Pro 解锁完整预算表」+ 按钮

### 3.7 Tab Bar（底部导航）

| 项 | 图标 | 文案 |
|----|------|------|
| 1 | home | 首页 |
| 2 | chart | 看板 |
| 3 | user | 我的 |

高度 50px + safe-area；选中色 `#2D8B7A`。

## 五、交互状态说明

### 5.1 全局

| 状态 | 表现 |
|------|------|
| Loading | 骨架屏（卡片占位）或全屏 spinner |
| Empty | 插画 + 文案 + CTA（如「还未测算，去生成规划书」） |
| Error | Toast + 重试按钮 |
| 弱网 | 表单本地缓存，顶部黄条「网络恢复后将同步」 |

- 最后一步前不可跳过必填
